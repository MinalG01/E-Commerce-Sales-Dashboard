# 📊 E-Commerce Sales Dashboard | Power BI

## 📌 Project Overview
This project is an interactive Power BI dashboard developed to analyze e-commerce sales performance. The dashboard provides insights into sales, profit, customer behavior, product performance, and regional trends to support data-driven business decisions.

## 🎯 Objective
- Analyze sales and profit performance.
- Identify top-performing products and categories.
- Track customer purchasing trends.
- Compare regional sales performance.
- Monitor key business KPIs.

## 🛠️ Tools & Technologies
- Power BI
- Power Query
- DAX
- Microsoft Excel

## 📂 Dataset
- Source: Public E-commerce Dataset (Kaggle)
- Data Cleaning: Removed duplicates, handled missing values, corrected data types, and transformed data using Power Query.

## 📊 Dashboard Features
- Sales Overview
- Profit Analysis
- Category & Sub-Category Performance
- Customer Analysis
- Regional Sales Analysis
- Interactive Filters & Slicers
- KPI Cards

## 📈 Key Insights
- Technology category generated the highest sales.
- Smartphones and Copiers were among the top revenue contributors.
- Some products, such as Tables, generated losses despite strong sales.
- A small group of customers contributed significantly to total sales.
- Regional performance varied, highlighting opportunities for business growth.
